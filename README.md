# xqz-web 技术知识库

基于 Obsidian 构建的系统化技术知识库，涵盖 Web 全栈开发核心技术栈。

## 写作规范

- **3W 原则**：每篇文章按 What → Why → How 组织
- **深度适中**：每篇 ≥6000 字，含 ≥3 个代码示例、≥2 个对比表格
- **面试模块**：每篇末尾 8 道精选面试题（含精炼答案）
- **双链导航**：各技术栈有 `_索引.md` 导航页，文章间通过 Obsidian 双链互联

## 使用方式

1. 用 Obsidian 打开本仓库根目录即可使用
2. 各技术栈入口：`_索引.md`
3. 通过 Obsidian 关系图谱、反向链接、搜索功能浏览知识网络
