# 统计分析与假设检验

## What — 统计分析与假设检验是什么

统计分析是从数据中提取信息、做出推断的方法论体系，分为描述统计（总结数据特征）和推断统计（从样本推断总体）。假设检验是推断统计的核心工具，通过设定零假设和备择假设，计算检验统计量和 p 值，判断样本证据是否足以拒绝零假设。

Python 统计分析的工具链：

| 工具 | 定位 | 说明 |
|------|------|------|
| scipy.stats | 核心统计库 | 分布、检验、描述统计 |
| statsmodels | 计量经济学 | 回归、时间序列、方差分析 |
| pingouin | 现代统计库 | 效应量、贝叶斯、重复测量 |
| numpy | 数值计算 | 均值、方差、协方差 |
| [[Pandas基础]] | 数据操作 | describe、value_counts |

```python
import numpy as np
from scipy import stats

# 描述统计：总结数据特征
data = np.array([23, 25, 28, 30, 22, 27, 24, 29, 26, 31])
print(f"均值: {np.mean(data):.2f}")
print(f"中位数: {np.median(data):.2f}")
print(f"标准差: {np.std(data, ddof=1):.2f}")  # ddof=1 样本标准差
print(f"偏度: {stats.skew(data):.3f}")
print(f"峰度: {stats.kurtosis(data):.3f}")

# 推断统计：从样本推断总体
# 单样本 t 检验：检验均值是否等于某个值
t_stat, p_value = stats.ttest_1samp(data, popmean=25)
print(f"t统计量: {t_stat:.3f}, p值: {p_value:.3f}")
```

## Why — 为什么需要假设检验

### 数据驱动的决策需要统计严谨性

仅看样本均值大小不能判断差异是否真实存在——抽样波动可能导致虚假差异。假设检验量化了"观察到的差异由随机波动产生的概率"，为决策提供统计依据。

| 场景 | 直觉判断 | 假设检验结论 |
|------|---------|-------------|
| A方案转化率5.2%，B方案5.5% | B更好 | p=0.35，差异不显著 |
| 新药有效率65%，旧药60% | 新药更好 | p=0.02，差异显著 |
| 两个班的平均分差3分 | 有差异 | 需看样本量和方差 |

### 假设检验的核心框架

```
1. 提出假设
   H₀（零假设）：无差异/无效果/符合分布
   H₁（备择假设）：有差异/有效果/不符合分布

2. 选择检验方法（根据数据类型和分布假设）

3. 计算检验统计量和 p 值

4. 做出决策
   p < α（通常0.05）→ 拒绝 H₀
   p ≥ α → 不能拒绝 H₀（不等于"接受 H₀"）
```

## How — 如何进行统计分析

### 1. 描述统计

```python
import numpy as np
from scipy import stats
import pandas as pd

rng = np.random.default_rng(42)
data = rng.normal(loc=100, scale=15, size=500)

# === 集中趋势 ===
print(f"均值: {np.mean(data):.2f}")
print(f"中位数: {np.median(data):.2f}")
print(f"众数: {stats.mode(data, keepdims=True)[0][0]:.2f}")  # 连续数据众数意义不大
print(f"截尾均值(10%): {stats.trim_mean(data, 0.1):.2f}")   # 去掉两端10%

# === 离散程度 ===
print(f"方差: {np.var(data, ddof=1):.2f}")        # 样本方差
print(f"标准差: {np.std(data, ddof=1):.2f}")       # 样本标准差
print(f"极差: {np.ptp(data):.2f}")                  # max - min
print(f"IQR: {np.percentile(data, 75) - np.percentile(data, 25):.2f}")
print(f"变异系数: {stats.variation(data):.4f}")     # std/mean

# === 分布形状 ===
print(f"偏度: {stats.skew(data):.3f}")      # >0右偏，<0左偏
print(f"峰度: {stats.kurtosis(data):.3f}")  # >0尖峰，<0平峰（超额峰度）

# === 分位数 ===
percentiles = np.percentile(data, [25, 50, 75, 90, 95, 99])
for p, v in zip([25, 50, 75, 90, 95, 99], percentiles):
    print(f"P{p}: {v:.2f}")

# === Pandas describe ===
df = pd.DataFrame({'score': data, 'group': rng.choice(['A', 'B'], 500)})
print(df.describe())
print(df.groupby('group')['score'].describe())

# === 相关性 ===
x = rng.normal(0, 1, 100)
y = 0.8 * x + rng.normal(0, 0.6, 100)

# Pearson 相关系数（线性关系）
r, p = stats.pearsonr(x, y)
print(f"Pearson r={r:.3f}, p={p:.4f}")

# Spearman 秩相关（单调关系）
rho, p = stats.spearmanr(x, y)
print(f"Spearman rho={rho:.3f}, p={p:.4f}")

# Kendall 秩相关（小样本、有结值）
tau, p = stats.kendalltau(x, y)
print(f"Kendall tau={tau:.3f}, p={p:.4f}")
```

| 指标 | 衡量内容 | 适用场景 |
|------|---------|---------|
| 均值 | 集中趋势 | 正态分布数据 |
| 中位数 | 集中趋势 | 偏态/有离群值 |
| 标准差 | 离散程度 | 与均值配合 |
| IQR | 离散程度 | 偏态/有离群值 |
| 偏度 | 对称性 | 分布形态诊断 |
| 峰度 | 尾部厚度 | 异常值风险评估 |
| Pearson r | 线性相关 | 连续变量、正态 |
| Spearman ρ | 单调相关 | 等级/非线性 |

### 2. 概率分布

scipy.stats 包含 100+ 连续和离散分布，每个分布都提供 pdf/pmf、cdf、ppf、rvs 等方法。

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# === 常用连续分布 ===

# 正态分布
norm = stats.norm(loc=0, scale=1)
print(f"PDF(0) = {norm.pdf(0):.4f}")      # 概率密度
print(f"CDF(1.96) = {norm.cdf(1.96):.4f}") # P(X ≤ 1.96) ≈ 0.975
print(f"PPF(0.975) = {norm.ppf(0.975):.4f}") # 反函数

# t 分布（小样本）
t_dist = stats.t(df=10)
print(f"t(10) 的 95% 分位点: {t_dist.ppf(0.975):.4f}")

# 卡方分布
chi2 = stats.chi2(df=5)
print(f"χ²(5) 的 95% 分位点: {chi2.ppf(0.95):.4f}")

# F 分布
f_dist = stats.f(dfn=3, dfd=20)
print(f"F(3,20) 的 95% 分位点: {f_dist.ppf(0.95):.4f}")

# 指数分布
exp_dist = stats.expon(scale=1/2)  # λ=2

# === 常用离散分布 ===

# 二项分布
binom = stats.binom(n=10, p=0.3)
print(f"B(10,0.3) P(X=3) = {binom.pmf(3):.4f}")
print(f"B(10,0.3) P(X≤3) = {binom.cdf(3):.4f}")

# 泊松分布
poisson = stats.poisson(mu=5)
print(f"Poi(5) P(X=3) = {poisson.pmf(3):.4f}")

# === 分布拟合 ===
# 用极大似然估计拟合正态分布
data = rng.normal(100, 15, 1000)
loc, scale = stats.norm.fit(data)
print(f"拟合: μ={loc:.2f}, σ={scale:.2f}")

# === 可视化分布 ===
fig, axes = plt.subplots(2, 2, figsize=(12, 8))

x = np.linspace(-4, 4, 200)
# 正态 vs t
axes[0, 0].plot(x, stats.norm.pdf(x), label='N(0,1)')
axes[0, 0].plot(x, stats.t(df=3).pdf(x), label='t(3)')
axes[0, 0].plot(x, stats.t(df=10).pdf(x), label='t(10)')
axes[0, 0].set_title('正态 vs t 分布')
axes[0, 0].legend()

# 卡方
x_chi2 = np.linspace(0.01, 20, 200)
for df in [2, 5, 10]:
    axes[0, 1].plot(x_chi2, stats.chi2(df=df).pdf(x_chi2), label=f'χ²({df})')
axes[0, 1].set_title('卡方分布')
axes[0, 1].legend()

# 二项分布
x_bin = np.arange(0, 16)
axes[1, 0].bar(x_bin - 0.2, stats.binom(15, 0.3).pmf(x_bin), width=0.4, label='B(15,0.3)')
axes[1, 0].bar(x_bin + 0.2, stats.binom(15, 0.5).pmf(x_bin), width=0.4, label='B(15,0.5)')
axes[1, 0].set_title('二项分布')
axes[1, 0].legend()

# 指数分布
x_exp = np.linspace(0, 5, 200)
for lam in [0.5, 1, 2]:
    axes[1, 1].plot(x_exp, stats.expon(scale=1/lam).pdf(x_exp), label=f'λ={lam}')
axes[1, 1].set_title('指数分布')
axes[1, 1].legend()

plt.tight_layout()
plt.show()
```

| 分布 | 参数 | 典型用途 |
|------|------|---------|
| norm(μ, σ) | 均值、标准差 | 自然现象、中心极限 |
| t(df) | 自由度 | 小样本均值检验 |
| chi2(df) | 自由度 | 方差检验、拟合优度 |
| F(dfn, dfd) | 两个自由度 | 方差分析、模型比较 |
| binom(n, p) | 试验次数、概率 | 成功次数 |
| poisson(λ) | 期望次数 | 稀有事件计数 |
| expon(λ) | 速率 | 等待时间 |

### 3. 参数检验

参数检验假设数据来自某个已知分布（通常为正态），是使用最广的检验方法。

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(42)

# === 正态性检验（前提检验） ===
data = rng.normal(0, 1, 100)

# Shapiro-Wilk 检验（推荐，n < 5000）
stat, p = stats.shapiro(data)
print(f"Shapiro-Wilk: W={stat:.4f}, p={p:.4f}")
# p > 0.05 → 不能拒绝正态假设

# D'Agostino-Pearson 检验（n > 20）
stat, p = stats.normaltest(data)
print(f"D'Agostino: K2={stat:.4f}, p={p:.4f}")

# Kolmogorov-Smirnov 检验
stat, p = stats.kstest(data, 'norm')
print(f"KS: D={stat:.4f}, p={p:.4f}")

# === 单样本 t 检验 ===
# 检验样本均值是否等于理论值
scores = rng.normal(72, 8, 30)
t_stat, p = stats.ttest_1samp(scores, popmean=70)
print(f"\n单样本t检验: t={t_stat:.3f}, p={p:.4f}")
# p < 0.05 → 均值显著不等于70

# === 独立样本 t 检验 ===
# 检验两组均值是否相等
group_a = rng.normal(75, 10, 40)
group_b = rng.normal(80, 10, 40)

# 等方差 t 检验
t_stat, p = stats.ttest_ind(group_a, group_b, equal_var=True)
print(f"等方差t检验: t={t_stat:.3f}, p={p:.4f}")

# Welch t 检验（不等方差，推荐默认使用）
t_stat, p = stats.ttest_ind(group_a, group_b, equal_var=False)
print(f"Welch t检验: t={t_stat:.3f}, p={p:.4f}")

# 先做方差齐性检验（Levene 检验）
lev_stat, lev_p = stats.levene(group_a, group_b)
print(f"Levene: F={lev_stat:.3f}, p={lev_p:.4f}")
# p > 0.05 → 方差齐性，可用等方差 t 检验

# === 配对 t 检验 ===
# 同一组对象前后测量
before = rng.normal(70, 8, 25)
after = before + rng.normal(5, 4, 25)  # 假设有提升

t_stat, p = stats.ttest_rel(before, after)
print(f"\n配对t检验: t={t_stat:.3f}, p={p:.4f}")

# 效应量：Cohen's d
diff = after - before
cohens_d = np.mean(diff) / np.std(diff, ddof=1)
print(f"Cohen's d = {cohens_d:.3f}")

# === z 检验（大样本） ===
# scipy.stats 没有直接 z 检验，手动计算
from statsmodels.stats.weightstats import ztest
data = rng.normal(100, 15, 200)
z_stat, p = ztest(data, value=100)
print(f"z检验: z={z_stat:.3f}, p={p:.4f}")
```

| 检验 | 零假设 | 适用场景 | 前提条件 |
|------|--------|---------|---------|
| Shapiro-Wilk | 数据服从正态 | 正态性检验 | n < 5000 |
| 单样本t | μ = μ₀ | 与标准值比较 | 正态性 |
| 独立样本t | μ₁ = μ₂ | 两组比较 | 正态性、独立性 |
| Welch t | μ₁ = μ₂ | 两组不等方差 | 正态性、独立性 |
| 配对t | μ_diff = 0 | 前后/配对比较 | 差值正态性 |
| Levene | σ₁² = σ₂² | 方差齐性检验 | 独立性 |

### 4. 方差分析（ANOVA）

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(42)

# === 单因素方差分析 ===
# 检验三组均值是否相等
group_a = rng.normal(70, 10, 30)
group_b = rng.normal(75, 10, 30)
group_c = rng.normal(80, 10, 30)

F_stat, p = stats.f_oneway(group_a, group_b, group_c)
print(f"单因素ANOVA: F={F_stat:.3f}, p={p:.6f}")
# p < 0.05 → 至少有一组均值显著不同

# 事后检验：哪两组之间有差异
from statsmodels.stats.multicomp import pairwise_tukeyhsd
import pandas as pd

data = np.concatenate([group_a, group_b, group_c])
labels = ['A'] * 30 + ['B'] * 30 + ['C'] * 30

tukey = pairwise_tukeyhsd(data, labels, alpha=0.05)
print(tukey)

# === 双因素方差分析 ===
import statsmodels.api as sm
from statsmodels.formula.api import ols

df = pd.DataFrame({
    'score': rng.normal(0, 1, 120),
    'method': np.repeat(['M1', 'M2', 'M3'], 40),
    'time': np.tile(['morning', 'afternoon'] * 3, 20)[:120]
})
df['score'] += df['method'].map({'M1': 10, 'M2': 15, 'M3': 12})
df['score'] += df['time'].map({'morning': 2, 'afternoon': 0})

model = ols('score ~ C(method) + C(time) + C(method):C(time)', data=df).fit()
anova_table = sm.stats.anova_lm(model, typ=2)
print(anova_table)

# === 重复测量方差分析 ===
# 使用 pingouin 库（更简洁）
# import pingouin as pg
# pg.rm_anova(data=df_rm, dv='score', within='time', subject='subject')
```

| 方法 | 零假设 | 适用场景 |
|------|--------|---------|
| 单因素ANOVA | 各组均值相等 | 3+组独立比较 |
| Tukey HSD | 两两均值相等 | ANOVA 事后检验 |
| 双因素ANOVA | 主效应和交互效应为零 | 两个因素同时影响 |
| 重复测量ANOVA | 各时间点均值相等 | 同一对象多次测量 |

### 5. 非参数检验

当数据不满足正态假设或为等级数据时，使用非参数检验。

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(42)

# === Mann-Whitney U 检验（对应独立样本t） ===
group_a = rng.exponential(scale=2, size=30)  # 非正态数据
group_b = rng.exponential(scale=3, size=30)

U_stat, p = stats.mannwhitneyu(group_a, group_b, alternative='two-sided')
print(f"Mann-Whitney U: U={U_stat:.1f}, p={p:.4f}")

# === Wilcoxon 符号秩检验（对应配对t） ===
before = rng.exponential(scale=5, size=25)
after = before + rng.normal(1, 2, 25)

W_stat, p = stats.wilcoxon(before, after)
print(f"Wilcoxon: W={W_stat:.1f}, p={p:.4f}")

# === Kruskal-Wallis H 检验（对应单因素ANOVA） ===
g1 = rng.exponential(1, 30)
g2 = rng.exponential(1.5, 30)
g3 = rng.exponential(2, 30)

H_stat, p = stats.kruskal(g1, g2, g3)
print(f"Kruskal-Wallis: H={H_stat:.3f}, p={p:.4f}")

# === Friedman 检验（对应重复测量ANOVA） ===
# 同一受试者在3种条件下的测量
cond1 = rng.normal(10, 3, 20)
cond2 = cond1 + rng.normal(2, 2, 20)
cond3 = cond1 + rng.normal(4, 2, 20)

F_stat, p = stats.friedmanchisquare(cond1, cond2, cond3)
print(f"Friedman: χ²={F_stat:.3f}, p={p:.4f}")

# === 卡方检验 ===
# 拟合优度检验：观测频率是否符合期望
observed = np.array([25, 30, 20, 25])
expected = np.array([25, 25, 25, 25])  # 均匀分布

chi2, p = stats.chisquare(observed, f_exp=expected)
print(f"\n卡方拟合优度: χ²={chi2:.3f}, p={p:.4f}")

# 独立性检验：两个分类变量是否关联
# 例：性别与偏好是否独立
observed_table = np.array([[30, 10, 15],   # 男性
                           [20, 25, 20]])   # 女性
chi2, p, dof, expected = stats.chi2_contingency(observed_table)
print(f"卡方独立性: χ²={chi2:.3f}, p={p:.4f}, df={dof}")

# Cramér's V（效应量）
n = observed_table.sum()
k = min(observed_table.shape) - 1
cramers_v = np.sqrt(chi2 / (n * k))
print(f"Cramér's V = {cramers_v:.3f}")

# === Fisher 精确检验（2×2 表，小样本） ===
table = np.array([[8, 2], [1, 5]])
odds_ratio, p = stats.fisher_exact(table)
print(f"Fisher精确检验: OR={odds_ratio:.2f}, p={p:.4f}")
```

| 非参数检验 | 对应参数检验 | 适用数据 |
|-----------|-------------|---------|
| Mann-Whitney U | 独立样本t | 两组非正态 |
| Wilcoxon | 配对t | 配对非正态 |
| Kruskal-Wallis | 单因素ANOVA | 3+组非正态 |
| Friedman | 重复测量ANOVA | 重复测量非正态 |
| 卡方拟合优度 | — | 单分类变量分布 |
| 卡方独立性 | — | 两个分类变量关联 |
| Fisher精确 | 卡方独立性 | 2×2 小样本 |

### 6. 效应量与统计功效

统计显著性（p值）不等于实际意义，效应量衡量差异的大小，统计功效衡量检测差异的能力。

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(42)

# === 效应量 ===

# Cohen's d（两组均值差异）
def cohens_d(g1, g2):
    n1, n2 = len(g1), len(g2)
    s_pooled = np.sqrt(((n1-1)*np.var(g1, ddof=1) + (n2-1)*np.var(g2, ddof=1)) / (n1+n2-2))
    return (np.mean(g1) - np.mean(g2)) / s_pooled

group_a = rng.normal(100, 15, 50)
group_b = rng.normal(110, 15, 50)
d = cohens_d(group_a, group_b)
print(f"Cohen's d = {d:.3f}")

# |d| < 0.2 微小, 0.2-0.5 小, 0.5-0.8 中, > 0.8 大

# 相关性效应量 (r²)
r, _ = stats.pearsonr(group_a, group_b)
r_squared = r ** 2
print(f"r² = {r_squared:.3f}")

# === 统计功效分析 ===
# 使用 statsmodels
from statsmodels.stats.power import TTestIndPower

power_analysis = TTestIndPower()

# 计算所需样本量
# 效应量 d=0.5（中等），α=0.05，功效=0.8
n_required = power_analysis.solve_power(effect_size=0.5, alpha=0.05, power=0.8)
print(f"所需样本量（每组）: {n_required:.0f}")

# 计算功效
power = power_analysis.solve_power(effect_size=0.5, alpha=0.05, nobs1=30)
print(f"功效（n=30）: {power:.3f}")

# 功效曲线
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(8, 5))
power_analysis.plot_power(
    dep_var='nobs',
    nobs=range(10, 200),
    effect_size=[0.2, 0.5, 0.8],
    alpha=0.05,
    ax=ax
)
ax.set_title('统计功效 vs 样本量')
plt.show()

# === 多重比较校正 ===
# 同时做多个检验时，假阳性率膨胀
from statsmodels.stats.multitest import multipletests

# 10 个 t 检验的 p 值
p_values = [0.003, 0.015, 0.042, 0.08, 0.12, 0.21, 0.35, 0.48, 0.62, 0.78]

# Bonferroni 校正（最保守）
reject, p_corrected, _, _ = multipletests(p_values, alpha=0.05, method='bonferroni')
print(f"Bonferroni 校正后显著: {reject}")
print(f"校正后 p 值: {p_corrected}")

# Benjamini-Hochberg 校正（FDR 控制，更常用）
reject, p_corrected, _, _ = multipletests(p_values, alpha=0.05, method='fdr_bh')
print(f"BH 校正后显著: {reject}")
print(f"校正后 p 值: {p_corrected}")
```

### 7. 置信区间

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(42)

# === 均值的置信区间 ===
data = rng.normal(100, 15, 50)
n = len(data)
mean = np.mean(data)
se = stats.sem(data)  # 标准误

# 95% 置信区间
ci_95 = stats.t.interval(0.95, df=n-1, loc=mean, scale=se)
print(f"均值 95% CI: [{ci_95[0]:.2f}, {ci_95[1]:.2f}]")

# 99% 置信区间
ci_99 = stats.t.interval(0.99, df=n-1, loc=mean, scale=se)
print(f"均值 99% CI: [{ci_99[0]:.2f}, {ci_99[1]:.2f}]")

# === 比例的置信区间 ===
# 1000 人中 650 人支持
from statsmodels.stats.proportion import proportion_confint
ci_low, ci_high = proportion_confint(650, 1000, alpha=0.05, method='wilson')
print(f"比例 95% CI: [{ci_low:.3f}, {ci_high:.3f}]")

# === Bootstrap 置信区间（非参数） ===
def bootstrap_ci(data, statistic=np.mean, n_boot=10000, alpha=0.05):
    rng = np.random.default_rng(42)
    boot_stats = []
    for _ in range(n_boot):
        sample = rng.choice(data, size=len(data), replace=True)
        boot_stats.append(statistic(sample))
    lower = np.percentile(boot_stats, 100 * alpha / 2)
    upper = np.percentile(boot_stats, 100 * (1 - alpha / 2))
    return lower, upper

ci = bootstrap_ci(data)
print(f"Bootstrap 95% CI: [{ci[0]:.2f}, {ci[1]:.2f}]")

# 中位数的 Bootstrap CI
ci_median = bootstrap_ci(data, statistic=np.median)
print(f"中位数 Bootstrap CI: [{ci_median[0]:.2f}, {ci_median[1]:.2f}]")
```

### 8. 回归诊断与统计建模

```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
from statsmodels.formula.api import ols
from scipy import stats

rng = np.random.default_rng(42)

# === 线性回归 ===
n = 100
x1 = rng.normal(0, 1, n)
x2 = rng.normal(0, 1, n)
y = 2 + 3 * x1 - 1.5 * x2 + rng.normal(0, 1, n)

df = pd.DataFrame({'y': y, 'x1': x1, 'x2': x2})

# 用公式接口
model = ols('y ~ x1 + x2', data=df).fit()
print(model.summary())

# 提取关键统计量
print(f"\nR² = {model.rsquared:.4f}")
print(f"调整R² = {model.rsquared_adj:.4f}")
print(f"F统计量 = {model.fvalue:.2f}, p = {model.f_pvalue:.6f}")
print(f"AIC = {model.aic:.2f}")
print(f"BIC = {model.bic:.2f}")

# 残差分析
residuals = model.resid
print(f"残差均值: {residuals.mean():.6f}")  # 应接近0
print(f"残差正态性 (Shapiro): p={stats.shapiro(residuals)[1]:.4f}")

# Durbin-Watson 检验（自相关）
from statsmodels.stats.stattools import durbin_watson
dw = durbin_watson(residuals)
print(f"Durbin-Watson: {dw:.4f}")  # 接近2为无自相关

# 多重共线性（VIF）
from statsmodels.stats.outliers_influence import variance_inflation_factor
X = sm.add_constant(df[['x1', 'x2']])
for i, col in enumerate(['const', 'x1', 'x2']):
    vif = variance_inflation_factor(X.values, i)
    print(f"VIF({col}) = {vif:.2f}")  # >10 表示严重共线性

# === 回归预测与置信区间 ===
predictions = model.get_prediction(df)
pred_df = predictions.summary_frame(alpha=0.05)
print(pred_df[['mean', 'mean_ci_lower', 'mean_ci_upper']].head())
```

## 面试题

### 1. p 值的含义是什么？p=0.03 代表什么？

p 值是在零假设成立的前提下，观察到当前或更极端结果的概率。p=0.03 意味着：如果零假设为真，仅有 3% 的概率会观察到当前数据这么极端或更极端的结果。这不意味着零假设为假的概率是 97%，也不意味着效应量很大。p 值只衡量证据强度，不衡量效应大小。小 p 值可能是大样本下微不足道差异的结果，大 p 值可能是小样本下真实差异未被检测到。

### 2. 第一类错误和第二类错误的关系？

第一类错误（α）是零假设为真时拒绝它（假阳性），第二类错误（β）是零假设为假时未拒绝它（假阴性）。统计功效 = 1 - β。在其他条件不变时，降低 α 会增大 β（更难拒绝 H₀），反之亦然。通过增大样本量可以同时降低两者。实践中 α 通常设为 0.05，功效目标为 0.8。A/B 测试中 α 控制误判新方案更好的概率，β 控制漏判新方案更好的概率。

### 3. 什么时候用参数检验，什么时候用非参数检验？

参数检验前提：数据近似正态、方差齐性、独立采样。满足前提时参数检验功效更高（更容易检测到真实差异）。非参数检验不假设数据分布，基于秩次，适用于：数据严重偏态或有大离群值、等级/有序数据、样本量极小（n < 30）、无法做正态变换。经验法则：先用 Shapiro-Wilk 检验正态性，通过则参数检验，不通过则非参数检验。但若样本量很大（n > 200），中心极限定理保证均值近似正态，参数检验仍然适用。

### 4. 置信区间和假设检验有什么关系？

两者是对偶的：如果 95% 置信区间不包含零假设值，则在 α=0.05 水平下拒绝零假设。置信区间比 p 值更有信息量——它不仅告诉你"是否显著"，还告诉你效应量的范围和精度。例如均值差 95% CI [0.5, 3.2] 表示差异显著且效应量可能在小到中等之间；CI [-0.3, 4.0] 即使 p≈0.05 也说明效应很不精确。报告结果时建议同时给出效应量和置信区间，而非仅 p 值。

### 5. 什么是多重比较问题？如何解决？

同时进行多次假设检验时，至少一个假阳性的概率膨胀。做 k 次独立检验，整体假阳性率 = 1-(1-α)^k，10次检验后约 40%。解决方案：（1）Bonferroni 校正——将 α 除以 k，最保守，保证整体 α 但可能过度保守；（2）Benjamini-Hochberg（FDR 控制）——控制假发现率，在基因表达等大量检验场景更实用；（3）Tukey HSD——专门用于 ANOVA 事后两两比较；（4）Dunnett——与对照组比较的专用校正。选择原则：少量比较用 Bonferroni，大量比较用 BH，ANOVA 事后用 Tukey。

### 6. ANOVA 和多次 t 检验有什么区别？

ANOVA 整体检验"所有组均值是否相等"，控制整体 α 在一次检验内；3 组做 3 次 t 检验，整体 α 膨胀到约 14%（1-(1-0.05)³）。ANOVA 只告诉你"至少有两组不同"，不告诉你具体哪两组——需要事后检验（Tukey HSD、Bonferroni 等）进行两两比较。另外 ANOVA 的 F 检验对均值差异的方向不敏感（双侧），比多次 t 检验的方向性重复更合理。4 组及以上必须用 ANOVA + 事后检验，而非多次 t 检验。

### 7. 效应量为什么比 p 值更重要？

p 值受样本量影响极大——大样本下极微小的差异也能显著，小样本下巨大的差异也可能不显著。效应量衡量差异的实际大小，与样本量无关。一个 d=0.1（微小效应）的结果可能 p<0.001（大样本），但实际意义可能不大；一个 d=0.9（大效应）的结果可能 p=0.08（小样本），但实际可能很重要。APA 和越来越多的期刊要求报告效应量。最佳实践：设计研究时基于期望效应量计算所需样本量，报告结果时同时给出 p 值、效应量和置信区间。

### 8. Bootstrap 方法有什么优势？

Bootstrap 通过有放回重采样来估计统计量的分布，无需假设数据服从特定分布。优势：（1）非参数——不需要正态假设或其他分布假设；（2）通用——适用于任何统计量（均值、中位数、相关系数、模型参数等）；（3）简单——概念直观，实现简单；（4）稳健——对离群值和偏态分布适应性好。局限：（1）计算量大（需数千次重采样）；（2）对极端小样本（n < 10）效果不好；（3）不能创造信息——如果样本本身有偏，Bootstrap 也有偏。适合场景：统计量分布难以解析推导、数据不满足参数假设、需要稳健的置信区间估计。
