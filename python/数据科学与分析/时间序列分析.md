# 时间序列分析

## What — 时间序列分析是什么

时间序列是按时间顺序排列的数据点序列，时间序列分析旨在从数据中提取趋势、季节性、周期性和噪声成分，并基于历史模式进行预测。Python 时间序列生态由 [[Pandas基础]]（时间索引与重采样）、statsmodels（统计建模）和专有库（Prophet、Darts）构成。

核心概念：

| 概念 | 说明 | 示例 |
|------|------|------|
| 趋势 | 长期上升/下降方向 | GDP 逐年增长 |
| 季节性 | 固定周期内的重复模式 | 冰淇淋销量夏高冬低 |
| 周期性 | 非固定周期的波动 | 经济周期（2-10年） |
| 平稳性 | 统计特性不随时间变化 | 白噪声 |
| 自相关 | 当前值与历史值的相关性 | 今日股价与昨日相关 |

```python
import pandas as pd
import numpy as np

# 创建时间序列
dates = pd.date_range('2023-01-01', periods=365, freq='D')
ts = pd.Series(
    np.random.default_rng(42).normal(0, 1, 365).cumsum() + 50,
    index=dates
)
print(ts.head())
print(f"频率: {ts.index.freq}")
```

## Why — 为什么需要时间序列分析

### 时间序列的独特性

时间序列与横截面数据（同一时间点的多个个体）不同，数据点之间存在时间依赖——违反了大多数统计方法"独立同分布"的假设。必须使用专门的方法来建模时间依赖结构。

| 对比项 | 横截面数据 | 时间序列 |
|--------|-----------|---------|
| 数据点关系 | 独立 | 时间依赖 |
| 核心假设 | i.i.d. | 自相关 |
| 核心任务 | 分类/回归 | 预测/分解 |
| 典型方法 | 线性回归、随机森林 | ARIMA、Prophet |
| 评估方式 | 随机划分 | 时序划分 |

### 应用场景

金融（股价预测）、销售（需求预测）、运维（容量规划）、气象（天气预报）、物联网（异常检测）、医疗（生理信号分析）。

## How — 如何进行时间序列分析

### 1. Pandas 时间序列基础

Pandas 是 Python 时间序列操作的核心工具，提供时间索引、重采样、滚动窗口等功能。

```python
import pandas as pd
import numpy as np

rng = np.random.default_rng(42)

# === 时间索引创建 ===

# DatetimeIndex
dates = pd.date_range('2023-01-01', periods=100, freq='D')
ts = pd.Series(rng.standard_normal(100).cumsum(), index=dates)

# PeriodIndex（时期）
periods = pd.period_range('2023-01', periods=12, freq='M')

# 常用频率字符串
# 'D' 日, 'B' 工作日, 'W' 周, 'M' 月末, 'MS' 月初
# 'Q' 季末, 'Y' 年末, 'H' 小时, 'T/min' 分钟, 'S' 秒

# === 时间索引操作 ===
# 按日期切片
print(ts['2023-01'])          # 整个月
print(ts['2023-01-15':'2023-02-15'])  # 日期范围

# 按部分索引
print(ts.loc['2023'])         # 整年
print(ts.loc['2023-Q1'])     # 第一季度

# === 移动与滞后 ===
df = pd.DataFrame({'value': [1, 2, 3, 4, 5]}, index=pd.date_range('2023-01-01', periods=5))

df['lag1'] = df['value'].shift(1)      # 向后移1期（滞后）
df['lead1'] = df['value'].shift(-1)    # 向前移1期（领先）
df['diff1'] = df['value'].diff()       # 一阶差分
df['pct_change'] = df['value'].pct_change()  # 百分比变化

print(df)
#    value  lag1  lead1  diff1  pct_change
# 0      1   NaN    2.0    NaN         NaN
# 1      2   1.0    3.0    1.0    1.000000
# 2      3   2.0    4.0    1.0    0.500000
# 3      4   3.0    5.0    1.0    0.333333
# 4      5   4.0    NaN    1.0    0.250000

# === 重采样 ===
# 高频 → 低频（降采样）
hourly = pd.Series(rng.standard_normal(720).cumsum(),
                    index=pd.date_range('2023-01-01', periods=720, freq='H'))

daily = hourly.resample('D').mean()       # 日均值
weekly = hourly.resample('W').agg({'mean', 'std', 'max'})  # 周统计

# 低频 → 高频（升采样）
daily_ts = pd.Series(rng.standard_normal(30),
                      index=pd.date_range('2023-01-01', periods=30, freq='D'))
hourly_filled = daily_ts.resample('H').ffill()  # 前向填充

# OHLC 重采样
prices = pd.Series(rng.standard_normal(720).cumsum() + 100,
                    index=pd.date_range('2023-01-01', periods=720, freq='H'))
ohlc = prices.resample('D').ohlc()  # 开高低收
```

### 2. 滚动窗口与指数平滑

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(42)
dates = pd.date_range('2022-01-01', periods=365, freq='D')
ts = pd.Series(rng.standard_normal(365).cumsum() + 50 + np.arange(365) * 0.05,
               index=dates)

# === 滚动窗口 ===
fig, ax = plt.subplots(figsize=(12, 5))

ax.plot(ts, alpha=0.4, label='原始')
ax.plot(ts.rolling(window=7).mean(), label='7日移动平均')
ax.plot(ts.rolling(window=30).mean(), label='30日移动平均')
ax.plot(ts.rolling(window=7).std(), label='7日滚动标准差', linestyle='--')
ax.legend()
ax.set_title('滚动窗口统计')
plt.show()

# 滚动窗口的其他统计
ts.rolling(7).median()      # 中位数
ts.rolling(7).max()         # 最大值
ts.rolling(7).quantile(0.75)  # 分位数
ts.rolling(7, min_periods=3).mean()  # 最少3个值

# === 指数加权移动平均（EWMA） ===
fig, ax = plt.subplots(figsize=(12, 5))

ax.plot(ts, alpha=0.3, label='原始')
ax.plot(ts.ewm(span=7).mean(), label='EWMA span=7')
ax.plot(ts.ewm(span=30).mean(), label='EWMA span=30')
ax.legend()
ax.set_title('指数加权移动平均')
plt.show()

# span vs com vs alpha
# alpha = 2 / (span + 1)  — span 直观
# alpha = 1 / (1 + com)    — com 居中质量
# alpha 直接指定平滑系数

# === 扩展窗口 ===
expanding_mean = ts.expanding().mean()  # 累积均值
expanding_max = ts.expanding().max()
```

| 方法 | 说明 | 权重 |
|------|------|------|
| rolling | 固定窗口 | 等权 |
| ewm | 指数衰减 | 近期权重大 |
| expanding | 扩展窗口 | 等权（累积） |

### 3. 时间序列分解

将时间序列拆分为趋势、季节性和残差三个成分。

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.seasonal import seasonal_decompose, STL

rng = np.random.default_rng(42)

# 生成含趋势+季节性+噪声的数据
n = 365 * 2
dates = pd.date_range('2022-01-01', periods=n, freq='D')
trend = np.linspace(50, 80, n)
seasonal = 10 * np.sin(2 * np.pi * np.arange(n) / 365)
noise = rng.standard_normal(n) * 3
ts = pd.Series(trend + seasonal + noise, index=dates)

# === 经典分解 ===
result = seasonal_decompose(ts, model='additive', period=365)

fig, axes = plt.subplots(4, 1, figsize=(12, 10))
result.observed.plot(ax=axes[0], title='原始')
result.trend.plot(ax=axes[1], title='趋势')
result.seasonal.plot(ax=axes[2], title='季节性')
result.resid.plot(ax=axes[3], title='残差')
plt.tight_layout()
plt.show()

# 乘法分解（季节性幅度随趋势增大）
# result = seasonal_decompose(ts, model='multiplicative', period=365)

# === STL 分解（更灵活） ===
stl = STL(ts, period=365, robust=True)
result_stl = stl.fit()

fig, axes = plt.subplots(4, 1, figsize=(12, 10))
result_stl.observed.plot(ax=axes[0], title='原始')
result_stl.trend.plot(ax=axes[1], title='趋势')
result_stl.seasonal.plot(ax=axes[2], title='季节性')
result_stl.resid.plot(ax=axes[3], title='残差')
plt.tight_layout()
plt.show()

# STL 优势：
# 1. 允许季节性随时间变化
# 2. robust=True 对离群值鲁棒
# 3. 可控制趋势和季节的平滑度
```

| 分解方法 | 季节性假设 | 优势 | 局限 |
|----------|-----------|------|------|
| 经典加法 | 固定幅度 | 简单 | 季节性不变 |
| 经典乘法 | 幅度随趋势变化 | 适合增长数据 | 需正值 |
| STL | 可变季节性 | 灵活、鲁棒 | 需要调参 |

### 4. 平稳性检验

ARIMA 等模型要求时间序列平稳（均值、方差不随时间变化）。实践中大部分时间序列非平稳，需要差分处理。

```python
import pandas as pd
import numpy as np
from statsmodels.tsa.stattools import adfuller, kpss

rng = np.random.default_rng(42)

# 非平稳序列（随机游走）
rw = pd.Series(rng.standard_normal(200).cumsum())

# 平稳序列
stationary = pd.Series(rng.standard_normal(200))

# === ADF 检验（Augmented Dickey-Fuller） ===
# H₀: 序列有单位根（非平稳）
# p < 0.05 → 拒绝 H₀ → 平稳

def adf_test(series, title=''):
    result = adfuller(series.dropna(), autolag='AIC')
    print(f'=== ADF 检验: {title} ===')
    print(f'ADF 统计量: {result[0]:.4f}')
    print(f'p 值: {result[1]:.4f}')
    print(f'使用的滞后阶数: {result[2]}')
    print(f'观测数: {result[3]}')
    for key, val in result[4].items():
        print(f'临界值 ({key}): {val:.4f}')
    print(f'结论: {"平稳" if result[1] < 0.05 else "非平稳"}')
    print()

adf_test(rw, '随机游走')
adf_test(stationary, '平稳序列')
adf_test(rw.diff().dropna(), '差分后的随机游走')

# === KPSS 检验（与 ADF 互补） ===
# H₀: 序列是平稳的（与 ADF 相反）
# p < 0.05 → 拒绝 H₀ → 非平稳

def kpss_test(series, title=''):
    result = kpss(series.dropna(), regression='c')  # 'c'=水平平稳, 'ct'=趋势平稳
    print(f'=== KPSS 检验: {title} ===')
    print(f'KPSS 统计量: {result[0]:.4f}')
    print(f'p 值: {result[1]:.4f}')
    print(f'使用的滞后阶数: {result[2]}')
    for key, val in result[3].items():
        print(f'临界值 ({key}): {val:.4f}')
    print(f'结论: {"非平稳" if result[1] < 0.05 else "平稳"}')
    print()

kpss_test(rw, '随机游走')
kpss_test(stationary, '平稳序列')

# === ADF + KPSS 联合判断 ===
# ADF拒绝 + KPSS不拒绝 → 严格平稳
# ADF不拒绝 + KPSS拒绝 → 非平稳，需差分
# 两者都拒绝/都不拒绝 → 不确定，需进一步分析
```

| 检验 | H₀ | p<0.05 结论 | 适用 |
|------|-----|------------|------|
| ADF | 有单位根（非平稳） | 平稳 | 大多数情况首选 |
| KPSS | 平稳 | 非平稳 | 互补验证 |
| PP | 有单位根 | 平稳 | ADF的稳健版本 |

### 5. ARIMA 模型

ARIMA（AutoRegressive Integrated Moving Average）是最经典的时序预测模型。

```
ARIMA(p, d, q):
  p — 自回归阶数（AR）：用过去 p 个值预测
  d — 差分次数（I）：使序列平稳
  q — 移动平均阶数（MA）：用过去 q 个误差预测
```

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.arima.model import ARIMA
from statsmodels.tsa.stattools import acf, pacf
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

rng = np.random.default_rng(42)

# 生成 AR(2) 数据
n = 300
ar_data = np.zeros(n)
for t in range(2, n):
    ar_data[t] = 0.6 * ar_data[t-1] - 0.3 * ar_data[t-2] + rng.standard_normal()

ts = pd.Series(ar_data, index=pd.date_range('2020-01-01', periods=n, freq='D'))

# === ACF 和 PACF 图（确定 p, q） ===
fig, axes = plt.subplots(1, 2, figsize=(14, 5))
plot_acf(ts, lags=30, ax=axes[0])
plot_pacf(ts, lags=30, ax=axes[1])
axes[0].set_title('ACF（自相关函数）')
axes[1].set_title('PACF（偏自相关函数）')
plt.tight_layout()
plt.show()

# ACF 拖尾 + PACF 截断(2) → AR(2)
# ACF 截断 + PACF 拖尾 → MA(q)
# 两者都拖尾 → ARMA

# === 拟合 ARIMA ===
# 划分训练/测试集
train = ts[:250]
test = ts[250:]

model = ARIMA(train, order=(2, 0, 0))  # AR(2)
result = model.fit()
print(result.summary())

# === 模型诊断 ===
fig = result.plot_diagnostics(figsize=(12, 8))
plt.tight_layout()
plt.show()

# Ljung-Box 检验：残差是否白噪声
from statsmodels.stats.diagnostic import acorr_ljungbox
lb_test = acorr_ljungbox(result.resid, lags=[10], return_df=True)
print(f"Ljung-Box p值: {lb_test['lb_pvalue'].values[0]:.4f}")
# p > 0.05 → 残差是白噪声 → 模型充分

# === 预测 ===
forecast = result.forecast(steps=50)

fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(train.index[-50:], train.values[-50:], label='训练集')
ax.plot(test.index, test.values, label='实际值')
ax.plot(test.index, forecast, label='预测值', linestyle='--')
ax.legend()
ax.set_title('ARIMA 预测')
plt.show()

# 带置信区间的预测
forecast_result = result.get_forecast(steps=50)
ci = forecast_result.conf_int(alpha=0.05)

fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(train.index[-50:], train.values[-50:], label='训练集')
ax.plot(test.index, test.values, label='实际值')
ax.plot(test.index, forecast_result.predicted_mean, label='预测', linestyle='--')
ax.fill_between(test.index, ci['lower y'], ci['upper y'], alpha=0.2)
ax.legend()
ax.set_title('ARIMA 预测 + 置信区间')
plt.show()

# === 自动确定 ARIMA 参数 ===
import pmdarima as pm

auto_model = pm.auto_arima(
    train,
    start_p=0, start_q=0,
    max_p=5, max_q=5,
    d=None,           # 自动确定差分次数
    seasonal=False,   # 非季节性
    stepwise=True,    # 逐步搜索
    information_criterion='aic',
    trace=True
)
print(f"最优参数: {auto_model.order}")
print(f"AIC: {auto_model.aic():.2f}")
```

| ACF 模式 | PACF 模式 | 模型 |
|----------|----------|------|
| 拖尾 | p 阶截断 | AR(p) |
| q 阶截断 | 拖尾 | MA(q) |
| 拖尾 | 拖尾 | ARMA(p,q) |
| 缓慢衰减 | — | 需要差分(ARIMA) |

### 6. SARIMA 季节性模型

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.statespace.sarimax import SARIMAX

rng = np.random.default_rng(42)

# 生成带季节性的数据
n = 365 * 3
dates = pd.date_range('2020-01-01', periods=n, freq='D')
trend = np.linspace(100, 130, n)
seasonal = 15 * np.sin(2 * np.pi * np.arange(n) / 365)
noise = rng.standard_normal(n) * 3
ts = pd.Series(trend + seasonal + noise, index=dates)

train = ts[:-90]
test = ts[-90:]

# SARIMA(p,d,q)(P,D,Q,s)
# s=365 季节周期，但 365 太大，改用月度数据
monthly = ts.resample('M').mean()
train_m = monthly[:-12]
test_m = monthly[-12:]

model = SARIMAX(
    train_m,
    order=(1, 1, 1),              # 非季节部分
    seasonal_order=(1, 1, 1, 12), # 季节部分: (P,D,Q,s)
    enforce_stationarity=False
)
result = model.fit(disp=False)
print(result.summary())

forecast = result.forecast(steps=12)

fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(train_m.index[-24:], train_m.values[-24:], label='训练')
ax.plot(test_m.index, test_m.values, label='实际')
ax.plot(test_m.index, forecast, label='预测', linestyle='--')
ax.legend()
ax.set_title('SARIMA 预测')
plt.show()

# 自动 SARIMA
import pmdarima as pm
auto_sarima = pm.auto_arima(
    train_m,
    start_p=0, start_q=0,
    max_p=3, max_q=3,
    start_P=0, start_Q=0,
    max_P=2, max_Q=2,
    m=12,              # 季节周期
    seasonal=True,
    d=1, D=1,
    stepwise=True,
    trace=True
)
print(f"最优参数: {auto_sarima.order} x {auto_sarima.seasonal_order}")
```

### 7. Prophet 与其他预测方法

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(42)

# 生成示例数据
n = 365 * 2
dates = pd.date_range('2022-01-01', periods=n, freq='D')
trend = np.linspace(100, 150, n)
seasonal = 20 * np.sin(2 * np.pi * np.arange(n) / 365)
weekly = 5 * np.sin(2 * np.pi * np.arange(n) / 7)
noise = rng.standard_normal(n) * 5
ts = pd.Series(trend + seasonal + weekly + noise, index=dates)

# === Prophet ===
from prophet import Prophet

df = pd.DataFrame({'ds': ts.index, 'y': ts.values})
train_df = df[:-90]

model = Prophet(
    yearly_seasonality=True,
    weekly_seasonality=True,
    daily_seasonality=False,
    changepoint_prior_scale=0.05  # 趋势灵活性
)
model.fit(train_df)

future = model.make_future_dataframe(periods=90)
forecast = model.predict(future)

# Prophet 自动分解
fig = model.plot_components(forecast)
plt.show()

fig = model.plot(forecast)
plt.show()

# === 指数平滑 ===
from statsmodels.tsa.holtwinters import ExponentialSmoothing

train = ts[:-90]
test = ts[-90:]

# Holt-Winters（加法季节性）
model = ExponentialSmoothing(
    train,
    trend='add',
    seasonal='add',
    seasonal_periods=365
)
result = model.fit()
forecast = result.forecast(90)

fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(train.index[-90:], train.values[-90:], label='训练')
ax.plot(test.index, test.values, label='实际')
ax.plot(test.index, forecast, label='Holt-Winters', linestyle='--')
ax.legend()
ax.set_title('Holt-Winters 指数平滑')
plt.show()

# === 简单基准模型 ===
# Naive: 用最后一个值预测
naive_forecast = np.full(90, train.iloc[-1])

# Seasonal Naive: 用去年同期的值预测
seasonal_naive = train.iloc[-365:-365+90].values

# Drift: 线性趋势
drift = np.linspace(train.iloc[-1], train.iloc[-1] + (train.iloc[-1] - train.iloc[0]) / len(train) * 90, 90)
```

| 方法 | 适用场景 | 优势 | 局限 |
|------|---------|------|------|
| ARIMA | 非季节性、平稳化后 | 理论成熟 | 需手动选参 |
| SARIMA | 季节性 | 建模完整 | 参数多 |
| Prophet | 多季节性+节假日 | 自动化、鲁棒 | 黑盒 |
| Holt-Winters | 趋势+季节 | 简单 | 季节固定 |
| auto_arima | 自动选参 | 省心 | 慢 |

### 8. 评估与实战

```python
import pandas as pd
import numpy as np

rng = np.random.default_rng(42)

# === 评估指标 ===
actual = rng.normal(100, 15, 50)
predicted = actual + rng.normal(0, 8, 50)

def mae(y_true, y_pred):
    return np.mean(np.abs(y_true - y_pred))

def mse(y_true, y_pred):
    return np.mean((y_true - y_pred) ** 2)

def rmse(y_true, y_pred):
    return np.sqrt(mse(y_true, y_pred))

def mape(y_true, y_pred):
    return np.mean(np.abs((y_true - y_pred) / y_true)) * 100

def smape(y_true, y_pred):
    return np.mean(2 * np.abs(y_true - y_pred) / (np.abs(y_true) + np.abs(y_pred))) * 100

print(f"MAE:  {mae(actual, predicted):.2f}")
print(f"MSE:  {mse(actual, predicted):.2f}")
print(f"RMSE: {rmse(actual, predicted):.2f}")
print(f"MAPE: {mape(actual, predicted):.2f}%")
print(f"sMAPE: {smape(actual, predicted):.2f}%")

# 不能随机划分，必须按时间顺序
from sklearn.model_selection import TimeSeriesSplit

ts_data = np.arange(100).astype(float)
tscv = TimeSeriesSplit(n_splits=5)

for fold, (train_idx, test_idx) in enumerate(tscv.split(ts_data)):
    print(f"Fold {fold+1}: 训练[{train_idx[0]}:{train_idx[-1]}] "
          f"测试[{test_idx[0]}:{test_idx[-1]}]")

# === 滚动预测（更真实） ===
def rolling_forecast(data, window_size, forecast_horizon, model_func):
    """滚动窗口预测：每次用 window_size 个历史值预测未来 forecast_horizon 步"""
    predictions = []
    for i in range(len(data) - window_size - forecast_horizon + 1):
        train = data[i:i+window_size]
        pred = model_func(train, forecast_horizon)
        predictions.append(pred)
    return np.array(predictions)

# 示例：用均值作为预测
def mean_forecast(train, h):
    return np.full(h, np.mean(train))

data = rng.standard_normal(200).cumsum()
preds = rolling_forecast(data, window_size=100, forecast_horizon=10,
                          model_func=mean_forecast)
print(f"滚动预测形状: {preds.shape}")  # (91, 10)

# === 异常检测 ===
# 基于滚动统计的异常检测
ts = pd.Series(rng.standard_normal(365).cumsum() + 50,
               index=pd.date_range('2023-01-01', periods=365))
ts.iloc[100] = 80  # 注入异常

rolling_mean = ts.rolling(7).mean()
rolling_std = ts.rolling(7).std()
upper = rolling_mean + 3 * rolling_std
lower = rolling_mean - 3 * rolling_std

anomalies = ts[(ts > upper) | (ts < lower)]
print(f"检测到 {len(anomalies)} 个异常点")
```

## 面试题

### 1. 时间序列的平稳性是什么？为什么重要？

平稳性指序列的统计特性（均值、方差、自协方差）不随时间平移而变化。严平稳要求联合分布不变，弱平稳（最常用）要求均值恒定、方差恒定、自协方差仅与滞后阶数有关。重要性：ARIMA 等模型的理论基础建立在平稳性假设上，非平稳序列的自相关不会衰减，模型参数估计不可靠，预测会发散。通过 ADF/KPSS 检验判断平稳性，非平稳序列通过差分（d 次）变为平稳。

### 2. ACF 和 PACF 的区别是什么？如何用于确定 ARIMA 参数？

ACF（自相关函数）测量 y_t 与 y_{t-k} 之间的总相关，包含间接相关；PACF（偏自相关函数）去除中间滞后的影响，只测量 y_t 与 y_{t-k} 之间的直接相关。参数确定规则：ACF 拖尾+PACF 在 p 阶截断→AR(p)；ACF 在 q 阶截断+PACF 拖尾→MA(q)；两者都拖尾→ARMA。实际中推荐用 `auto_arima` 基于 AIC/BIC 自动搜索最优参数，ACF/PACF 图作为辅助验证。

### 3. ARIMA 和 SARIMA 的区别？

ARIMA(p,d,q) 建模非季节性的自回归、差分和移动平均结构。SARIMA(p,d,q)(P,D,Q,s) 在 ARIMA 基础上增加了季节性部分：P 是季节自回归阶数、D 是季节差分次数、Q 是季节移动平均阶数、s 是季节周期（如12个月、4个季度）。季节差分是 y_t - y_{t-s}，消除固定周期的重复模式。如果数据有季节性但只用 ARIMA，残差会在季节滞后处显著自相关，模型不充分。

### 4. Prophet 的核心原理是什么？

Prophet（Facebook 开源）使用加法模型：y(t) = g(t) + s(t) + h(t) + ε(t)，其中 g(t) 是分段线性或逻辑增长趋势（带自动变点检测），s(t) 是傅里叶级数拟合的季节性（支持多年周期和周周期），h(t) 是节假日效应，ε(t) 是误差。优势：自动检测趋势变点、用 Laplace 先验控制变点稀疏性、对缺失值和异常值鲁棒、支持自定义节假日。适合日粒度、有强季节性和节假日效应的商业时间序列，不适合高频金融数据。

### 5. 时序交叉验证和普通交叉验证有什么区别？

普通 K-Fold 随机打乱数据，时序数据不能这样做——未来数据不能用于预测过去。TimeSeriesSplit 保持时间顺序：第1折用 [0:n1] 训练、[n1:n2] 测试；第2折用 [0:n2] 训练、[n2:n3] 测试，依此类推。更接近真实场景的是滚动窗口评估（expanding window 或 sliding window），每次预测后移动窗口。评估时序模型还应注意：不能只看整体指标，要分区间看预测误差是否随时间增长（预测衰减问题）。

### 6. 如何处理时间序列中的缺失值？

三种策略：（1）**前向填充** `ffill()`——用最近的历史值填充，适合短间隔缺失；（2）**插值** `interpolate(method='time')`——线性/样条插值，考虑时间间隔不均匀，效果更平滑；（3）**模型填补**——用 ARIMA 等模型预测缺失值，最精确但最复杂。选择原则：少量缺失用 ffill 或线性插值；有季节性的数据用季节性插值；缺失段较长用模型预测。避免用均值填充（破坏时间依赖）和后向填充（数据泄露）。

### 7. 什么是差分？一阶差分和二阶差分的区别？

差分是取相邻观测值的差：一阶差分 Δy_t = y_t - y_{t-1}，消除线性趋势；二阶差分 Δ²y_t = Δy_t - Δy_{t-1}，消除二次趋势。ARIMA 中的 d 参数是差分次数：d=0 原始序列已平稳，d=1 一阶差分后平稳（最常见），d=2 二阶差分后平稳（少见，通常过度差分）。过度差分的问题：引入不必要的噪声、降低预测精度、使 MA 多项式出现单位根。通过 ADF 检验确定最小差分次数。

### 8. MAPE 和 sMAPE 有什么局限性？

MAPE = mean(|y - ŷ| / |y|) 在 y 接近0时分母趋向0，值爆炸甚至无定义（y=0时），因此不适合包含0或接近0的数据（如温度、差额）。sMAPE 用 |y|+|ŷ| 作为分母缓解了这个问题，但仍不是对称的——过高预测和过低预测的惩罚不同。两者都是相对指标，不反映绝对误差大小。更好的选择：MASE（与基准模型的 MAE 之比）不受尺度影响且能处理零值；或直接报告 MAE + RMSE，一个可解释，一个惩罚大误差。
