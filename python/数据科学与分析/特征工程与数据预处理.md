# 特征工程与数据预处理

## What — 特征工程与数据预处理是什么

特征工程是将原始数据转化为更适合机器学习模型的特征的过程，数据预处理则是建模前的数据清洗和标准化步骤。两者紧密相关，共同决定模型性能的上限——"数据和特征决定了机器学习的上限，模型和算法只是逼近这个上限"。

核心流程：

| 步骤 | 说明 | 工具 |
|------|------|------|
| 数据清洗 | 缺失值、异常值、重复值 | [[Pandas基础]] |
| 数值变换 | 缩放、归一化、分布变换 | scikit-learn |
| 类别编码 | 标签编码、独热编码、目标编码 | scikit-learn / category_encoders |
| 特征构造 | 交叉特征、多项式特征、领域特征 | 手动 / Featuretools |
| 特征选择 | 过滤法、包裹法、嵌入法 | scikit-learn |
| 降维 | PCA、t-SNE、UMAP | scikit-learn |

```python
import numpy as np
import pandas as pd
from sklearn.preprocessing import StandardScaler, OneHotEncoder
from sklearn.impute import SimpleImputer

# 特征工程的基本流程
df = pd.DataFrame({
    'age': [25, 30, np.nan, 45, 50],
    'income': [50000, 60000, 75000, np.nan, 120000],
    'city': ['Beijing', 'Shanghai', 'Beijing', 'Guangzhou', 'Shanghai']
})

# 1. 缺失值填充
imputer = SimpleImputer(strategy='median')
df[['age', 'income']] = imputer.fit_transform(df[['age', 'income']])

# 2. 数值缩放
scaler = StandardScaler()
df[['age', 'income']] = scaler.fit_transform(df[['age', 'income']])

# 3. 类别编码
encoder = OneHotEncoder(sparse_output=False)
city_encoded = encoder.fit_transform(df[['city']])
print(df)
```

## Why — 为什么特征工程如此重要

### 垃圾进垃圾出

模型只能在特征空间内学习模式。特征质量差（含噪声、尺度不一致、信息冗余），再好的模型也无济于事。Kaggle 竞赛的实践经验：特征工程对排名的影响远大于模型选择。

| 场景 | 原始特征 | 工程后特征 | 效果 |
|------|---------|-----------|------|
| 房价预测 | 经纬度 | 到市中心的距离 | 线性可分 |
| 用户流失 | 注册日期 | 账户年龄（天） | 直接可用 |
| 信用评分 | 月收入/月支出 | 债务收入比 | 更强信号 |
| 时间序列 | 时间戳 | 星期几/是否节假日 | 捕捉周期性 |
| 文本分类 | 原始文本 | TF-IDF 向量 | 数值化 |

### scikit-learn 的 Transformer 架构

scikit-learn 的预处理组件遵循统一的 `fit` → `transform` 接口，可以和模型一起组成 Pipeline。

```python
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.impute import SimpleImputer
from sklearn.linear_model import LogisticRegression

# Pipeline 确保预处理参数只在训练集上 fit，避免数据泄露
pipe = Pipeline([
    ('imputer', SimpleImputer(strategy='median')),
    ('scaler', StandardScaler()),
    ('model', LogisticRegression())
])

pipe.fit(X_train, y_train)
score = pipe.score(X_test, y_test)
```

## How — 如何进行特征工程

### 1. 缺失值处理

```python
import numpy as np
import pandas as pd
from sklearn.impute import SimpleImputer, KNNImputer

rng = np.random.default_rng(42)

df = pd.DataFrame({
    'age': [25, 30, np.nan, 45, 50, np.nan, 35, 28],
    'income': [50000, 60000, 75000, np.nan, 120000, 80000, np.nan, 55000],
    'score': [85, 90, 78, 92, np.nan, 88, 76, np.nan],
    'category': ['A', 'B', 'A', np.nan, 'C', 'B', 'A', 'B']
})

# === 分析缺失模式 ===
print("缺失统计:")
print(df.isnull().sum())
print(f"\n缺失比例:\n{df.isnull().mean()}")

# 缺失相关性：一个字段的缺失是否与另一个字段相关
print(f"\n缺失相关性:\n{df.isnull().corr()}")

# === 删除法 ===
# 删除有缺失的行
df_drop_row = df.dropna()
# 删除有缺失的列
df_drop_col = df.dropna(axis=1)
# 删除全缺失的行
df_drop_all = df.dropna(how='all')
# 至少保留 N 个非缺失值
df_drop_thresh = df.dropna(thresh=3)

# === 常量填充 ===
# 均值/中位数/众数
imputer_mean = SimpleImputer(strategy='mean')
imputer_median = SimpleImputer(strategy='median')
imputer_mode = SimpleImputer(strategy='most_frequent')
imputer_const = SimpleImputer(strategy='constant', fill_value=0)

df[['age', 'income']] = imputer_median.fit_transform(df[['age', 'income']])

# === KNN 填充 ===
# 基于相似样本的值填充，更精确但更慢
knn_imputer = KNNImputer(n_neighbors=3)
df[['age', 'income', 'score']] = knn_imputer.fit_transform(df[['age', 'income', 'score']])

# === 分类变量缺失 ===
# 众数填充
df['category'] = df['category'].fillna(df['category'].mode()[0])
# 或作为单独类别
df['category'] = df['category'].fillna('Missing')

# === 标记缺失 ===
# 有时"是否缺失"本身就是有价值的特征
df['age_missing'] = df['age'].isnull().astype(int)

# === 时间序列缺失 ===
ts = pd.Series([1, np.nan, np.nan, 4, 5, np.nan, 7],
               index=pd.date_range('2023-01-01', periods=7))
ts_ffill = ts.ffill()           # 前向填充
ts_bfill = ts.bfill()           # 后向填充
ts_interp = ts.interpolate()    # 线性插值
ts_interp_time = ts.interpolate(method='time')  # 时间感知插值
```

| 方法 | 优点 | 缺点 | 适用 |
|------|------|------|------|
| 删除 | 简单 | 信息损失 | 缺失极少(<5%) |
| 均值/中位数 | 简单 | 降低方差 | 数值型、缺失少 |
| KNN | 保留数据结构 | 慢、需缩放 | 有意义的特征空间 |
| 插值 | 平滑 | 假设连续性 | 时间序列 |
| 标记缺失 | 保留信息 | 增加维度 | 缺失本身有意义 |

### 2. 数值特征变换

```python
import numpy as np
import pandas as pd
from sklearn.preprocessing import (
    StandardScaler, MinMaxScaler, RobustScaler,
    PowerTransformer, QuantileTransformer, PolynomialFeatures
)

rng = np.random.default_rng(42)

# 生成不同分布的数据
df = pd.DataFrame({
    'normal': rng.normal(50, 10, 500),
    'skewed': rng.exponential(20, 500),        # 右偏
    'bimodal': np.concatenate([rng.normal(20, 5, 250), rng.normal(60, 5, 250)]),
    'outlier': np.concatenate([rng.normal(0, 1, 490), rng.uniform(20, 30, 10)])
})

# === 标准化 / 归一化 ===

# StandardScaler: 均值0，标准差1 — 最常用
scaler = StandardScaler()
df_scaled = pd.DataFrame(scaler.fit_transform(df), columns=df.columns)
print(f"StandardScaler 均值: {df_scaled.mean().round(10).tolist()}")
print(f"StandardScaler 标准差: {df_scaled.std().round(2).tolist()}")

# MinMaxScaler: 缩放到 [0, 1]
mm_scaler = MinMaxScaler()
df_mm = pd.DataFrame(mm_scaler.fit_transform(df), columns=df.columns)
print(f"\nMinMax 范围: [{df_mm.min().min():.2f}, {df_mm.max().max():.2f}]")

# RobustScaler: 用中位数和IQR，对离群值鲁棒
rb_scaler = RobustScaler()
df_rb = pd.DataFrame(rb_scaler.fit_transform(df), columns=df.columns)

# 自定义范围
mm_custom = MinMaxScaler(feature_range=(-1, 1))

# === 分布变换 ===

# Yeo-Johnson 变换（处理偏态，支持正负值）
pt = PowerTransformer(method='yeo-johnson')
df_yj = pd.DataFrame(pt.fit_transform(df), columns=df.columns)

# Box-Cox 变换（仅正值）
pt_bc = PowerTransformer(method='box-cox')
df_bc = pd.DataFrame(pt_bc.fit_transform(df[['skewed']]), columns=['skewed'])

# 分位数变换（映射到均匀/正态分布）
qt = QuantileTransformer(output_distribution='normal', n_quantiles=300)
df_qt = pd.DataFrame(qt.fit_transform(df), columns=df.columns)

# 对数变换（手动）
df['skewed_log'] = np.log1p(df['skewed'])   # log(1+x)，处理0值

# === 多项式特征 ===
from sklearn.linear_model import LinearRegression

X = rng.uniform(-3, 3, (100, 1))
y = 0.5 * X.ravel()**2 - X.ravel() + rng.normal(0, 1, 100)

# 线性模型无法拟合二次关系
lr = LinearRegression()
lr.fit(X, y)
print(f"线性 R²: {lr.score(X, y):.4f}")

# 添加多项式特征
poly = PolynomialFeatures(degree=2, include_bias=False)
X_poly = poly.fit_transform(X)  # [x, x²]
lr.fit(X_poly, y)
print(f"多项式 R²: {lr.score(X_poly, y):.4f}")

# === 分箱（离散化） ===
from sklearn.preprocessing import KBinsDiscretizer

kb = KBinsDiscretizer(n_bins=5, encode='ordinal', strategy='quantile')
df['age_bin'] = kb.fit_transform(df[['normal']])

# 等宽分箱
df['age_equal'] = pd.cut(df['normal'], bins=5, labels=False)
# 等频分箱
df['age_quantile'] = pd.qcut(df['normal'], q=5, labels=False)
```

| 方法 | 公式 | 适用模型 | 对离群值 |
|------|------|---------|---------|
| StandardScaler | (x-μ)/σ | SVM、LR、NN | 敏感 |
| MinMaxScaler | (x-min)/(max-min) | NN、KNN | 极敏感 |
| RobustScaler | (x-median)/IQR | 有离群值时 | 鲁棒 |
| PowerTransformer | Yeo-Johnson/Box-Cox | 线性模型 | 较鲁棒 |
| QuantileTransformer | 映射到目标分布 | 非线性模型 | 鲁棒 |
| 对数变换 | log(1+x) | 右偏数据 | 压缩 |

### 3. 类别特征编码

```python
import numpy as np
import pandas as pd
from sklearn.preprocessing import LabelEncoder, OrdinalEncoder, OneHotEncoder

df = pd.DataFrame({
    'color': ['red', 'blue', 'green', 'red', 'blue'],
    'size': ['S', 'M', 'L', 'XL', 'M'],
    'city': ['Beijing', 'Shanghai', 'Guangzhou', 'Beijing', 'Shenzhen']
})

# === 标签编码 ===
# 无序类别 → 整数（引入虚假序关系，仅树模型可用）
le = LabelEncoder()
df['color_encoded'] = le.fit_transform(df['color'])
print(f"映射: {dict(zip(le.classes_, le.transform(le.classes_)))}")

# === 有序编码 ===
# 有序类别 → 保留顺序
size_order = ['S', 'M', 'L', 'XL']
oe = OrdinalEncoder(categories=[size_order])
df['size_encoded'] = oe.fit_transform(df[['size']]).astype(int)

# === 独热编码（One-Hot） ===
# 无序类别 → 二值列（最常用，但高基数时维度爆炸）
ohe = OneHotEncoder(sparse_output=False, drop='first')  # drop避免多重共线性
city_ohe = ohe.fit_transform(df[['city']])
print(f"独热编码列: {ohe.get_feature_names_out()}")

# pandas 方式（更直观）
df_ohe = pd.get_dummies(df, columns=['city'], drop_first=True)

# === 频率编码 ===
# 用类别出现频率替代类别值，适合高基数
freq = df['city'].value_counts(normalize=True)
df['city_freq'] = df['city'].map(freq)

# === 目标编码（Target Encoding） ===
# 用类别对应的目标变量均值替代，适合高基数+树模型
# 注意：必须在训练集上计算，否则数据泄露
from sklearn.model_selection import KFold

def target_encode(train_series, target, test_series=None, n_folds=5, smooth=0):
    """K-Fold 目标编码，防止过拟合"""
    global_mean = target.mean()
    encoded = pd.Series(index=train_series.index, dtype=float)

    kf = KFold(n_splits=n_folds, shuffle=True, random_state=42)
    for train_idx, val_idx in kf.split(train_series):
        fold_means = target.iloc[train_idx].groupby(train_series.iloc[train_idx]).mean()
        counts = train_series.iloc[train_idx].groupby(train_series.iloc[train_idx]).count()

        # 平滑：小样本类别向全局均值收缩
        smoothed = (fold_means * counts + global_mean * smooth) / (counts + smooth)
        encoded.iloc[val_idx] = train_series.iloc[val_idx].map(smoothed)

    encoded = encoded.fillna(global_mean)

    if test_series is not None:
        train_means = target.groupby(train_series).mean()
        counts = train_series.value_counts()
        smoothed = (train_means * counts + global_mean * smooth) / (counts + smooth)
        test_encoded = test_series.map(smoothed).fillna(global_mean)
        return encoded, test_encoded

    return encoded

# === 二值编码（Binary Encoding） ===
# 类别→整数→二进制→拆列，比独热更省维度
# pip install category_encoders
# from category_encoders import BinaryEncoder
# be = BinaryEncoder(cols=['city'])
# df_binary = be.fit_transform(df)

# === 编码方法选择 ===
```

| 编码方法 | 适用模型 | 高基数 | 有序性 | 维度 |
|----------|---------|--------|--------|------|
| LabelEncoder | 树模型 | 可以 | 无 | +1 |
| OrdinalEncoder | 树/线性 | 可以 | 保留 | +1 |
| OneHotEncoder | 线性/NN | 不适合 | 无 | +k-1 |
| 频率编码 | 树模型 | 适合 | 无 | +1 |
| 目标编码 | 树模型 | 适合 | 无 | +1 |
| BinaryEncoder | 线性/树 | 适合 | 无 | +log₂k |

### 4. 特征构造

```python
import numpy as np
import pandas as pd
from sklearn.preprocessing import PolynomialFeatures

rng = np.random.default_rng(42)

# === 数值特征构造 ===
df = pd.DataFrame({
    'length': rng.uniform(1, 10, 100),
    'width': rng.uniform(1, 10, 100),
    'height': rng.uniform(1, 5, 100),
    'weight': rng.uniform(0.5, 20, 100),
    'price': rng.uniform(10, 100, 100)
})

# 比值特征
df['aspect_ratio'] = df['length'] / df['width']
df['price_per_weight'] = df['price'] / df['weight']

# 交互特征
df['volume'] = df['length'] * df['width'] * df['height']
df['density'] = df['weight'] / df['volume']

# 多项式交互（自动生成所有组合）
poly = PolynomialFeatures(degree=2, interaction_only=True, include_bias=False)
X_poly = poly.fit_transform(df[['length', 'width', 'height']])
print(f"交互特征名: {poly.get_feature_names_out()}")

# === 时间特征构造 ===
df_time = pd.DataFrame({
    'timestamp': pd.date_range('2023-01-01', periods=365, freq='D'),
    'value': rng.standard_normal(365).cumsum()
})

df_time['year'] = df_time['timestamp'].dt.year
df_time['month'] = df_time['timestamp'].dt.month
df_time['day'] = df_time['timestamp'].dt.day
df_time['day_of_week'] = df_time['timestamp'].dt.dayofweek
df_time['day_of_year'] = df_time['timestamp'].dt.dayofyear
df_time['week_of_year'] = df_time['timestamp'].dt.isocalendar().week.astype(int)
df_time['is_weekend'] = (df_time['day_of_week'] >= 5).astype(int)
df_time['quarter'] = df_time['timestamp'].dt.quarter
df_time['is_month_start'] = df_time['timestamp'].dt.is_month_start.astype(int)
df_time['is_month_end'] = df_time['timestamp'].dt.is_month_end.astype(int)

# 周期编码（避免 12月和1月的距离问题）
df_time['month_sin'] = np.sin(2 * np.pi * df_time['month'] / 12)
df_time['month_cos'] = np.cos(2 * np.pi * df_time['month'] / 12)
df_time['dow_sin'] = np.sin(2 * np.pi * df_time['day_of_week'] / 7)
df_time['dow_cos'] = np.cos(2 * np.pi * df_time['day_of_week'] / 7)

# 节假日特征（需安装 holidays 库）
# import holidays
# cn_holidays = holidays.CN()
# df_time['is_holiday'] = df_time['timestamp'].dt.date.apply(lambda x: x in cn_holidays).astype(int)

# === 文本特征构造 ===
df_text = pd.DataFrame({
    'text': ['Hello World', 'Python is great', 'Feature engineering matters', 'Data science rocks']
})

df_text['char_count'] = df_text['text'].str.len()
df_text['word_count'] = df_text['text'].str.split().str.len()
df_text['avg_word_len'] = df_text['char_count'] / df_text['word_count']
df_text['has_number'] = df_text['text'].str.contains(r'\d').astype(int)
df_text['uppercase_ratio'] = df_text['text'].apply(lambda x: sum(c.isupper() for c in x) / len(x))

# === 聚合特征 ===
# 用户行为聚合
df_behavior = pd.DataFrame({
    'user_id': rng.choice(['A', 'B', 'C'], 100),
    'action': rng.choice(['click', 'view', 'purchase'], 100),
    'amount': rng.uniform(1, 100, 100)
})

agg_features = df_behavior.groupby('user_id').agg(
    total_amount=('amount', 'sum'),
    avg_amount=('amount', 'mean'),
    max_amount=('amount', 'max'),
    purchase_count=('action', lambda x: (x == 'purchase').sum()),
    action_diversity=('action', 'nunique'),
    action_count=('action', 'count')
)
print(agg_features)
```

### 5. 特征选择

```python
import numpy as np
import pandas as pd
from sklearn.feature_selection import (
    VarianceThreshold, SelectKBest, f_classif, mutual_info_classif,
    RFE, SelectFromModel
)
from sklearn.ensemble import RandomForestClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import cross_val_score

rng = np.random.default_rng(42)

# 生成数据：20个特征，其中5个有用
n = 500
X_useful = rng.standard_normal((n, 5))
y = (X_useful[:, 0] + 2 * X_useful[:, 1] - X_useful[:, 2] > 0).astype(int)
X_noise = rng.standard_normal((n, 15))
X = np.hstack([X_useful, X_noise])

# === 过滤法（Filter） ===

# 方差阈值：删除方差过小的特征
vt = VarianceThreshold(threshold=0.01)
X_vt = vt.fit_transform(X)
print(f"方差阈值后: {X.shape[1]} → {X_vt.shape[1]}")

# 单变量统计检验：选择与目标最相关的 K 个特征
skb_f = SelectKBest(f_classif, k=10)
X_f = skb_f.fit_transform(X, y)
print(f"F检验选择的特征: {skb_f.get_support(indices=True)}")

skb_mi = SelectKBest(mutual_info_classif, k=10)
X_mi = skb_mi.fit_transform(X, y)
print(f"互信息选择的特征: {skb_mi.get_support(indices=True)}")

# 相关性过滤：删除高度相关的特征
df_corr = pd.DataFrame(X)
corr_matrix = df_corr.corr().abs()
upper = corr_matrix.where(np.triu(np.ones(corr_matrix.shape), k=1).astype(bool))
high_corr_features = [col for col in upper.columns if any(upper[col] > 0.95)]
print(f"高相关特征数: {len(high_corr_features)}")

# === 包裹法（Wrapper） ===

# 递归特征消除（RFE）
lr = LogisticRegression(max_iter=1000, random_state=42)
rfe = RFE(lr, n_features_to_select=8, step=2)
X_rfe = rfe.fit_transform(X, y)
print(f"\nRFE 选择的特征: {rfe.get_support(indices=True)}")
print(f"特征排名: {rfe.ranking_}")

# === 嵌入法（Embedded） ===

# 基于 L1 正则化的特征选择
lr_l1 = LogisticRegression(penalty='l1', solver='saga', C=0.1, max_iter=2000, random_state=42)
sfm_l1 = SelectFromModel(lr_l1, threshold='median')
X_l1 = sfm_l1.fit_transform(X, y)
print(f"L1 选择的特征: {sfm_l1.get_support(indices=True)}")

# 基于树模型的特征重要性
rf = RandomForestClassifier(n_estimators=100, random_state=42)
sfm_rf = SelectFromModel(rf, threshold='median')
X_rf = sfm_rf.fit_transform(X, y)
print(f"RF 选择的特征: {sfm_rf.get_support(indices=True)}")

# 查看特征重要性排序
rf.fit(X, y)
importances = pd.Series(rf.feature_importances_, index=[f'f{i}' for i in range(20)])
print(f"\nRF 特征重要性 Top5:\n{importances.nlargest(5)}")

# === 排列重要性（更可靠） ===
from sklearn.inspection import permutation_importance

result = permutation_importance(rf, X, y, n_repeats=10, random_state=42, n_jobs=-1)
perm_importance = pd.Series(result.importances_mean, index=[f'f{i}' for i in range(20)])
print(f"\n排列重要性 Top5:\n{perm_importance.nlargest(5)}")
```

| 方法 | 类型 | 计算成本 | 考虑交互 | 过拟合风险 |
|------|------|---------|---------|-----------|
| 方差阈值 | 过滤 | 极低 | 否 | 无 |
| 统计检验 | 过滤 | 低 | 否 | 无 |

### 7. 异常值处理

```python
import numpy as np
import pandas as pd
from sklearn.ensemble import IsolationForest
from sklearn.neighbors import LocalOutlierFactor

rng = np.random.default_rng(42)

# 生成含异常值的数据
n = 200
X = rng.normal(0, 1, (n, 2))
X_outliers = rng.uniform(-8, 8, (10, 2))
X_all = np.vstack([X, X_outliers])

# === 统计方法 ===
# IQR 法
def detect_outliers_iqr(data, factor=1.5):
    Q1 = np.percentile(data, 25)
    Q3 = np.percentile(data, 75)
    IQR = Q3 - Q1
    lower = Q1 - factor * IQR
    upper = Q3 + factor * IQR
    return (data < lower) | (data > upper)

outliers_iqr = detect_outliers_iqr(X_all[:, 0])
print(f"IQR 检测异常: {outliers_iqr.sum()} 个")

# Z-score 法
def detect_outliers_zscore(data, threshold=3):
    z = (data - np.mean(data)) / np.std(data)
    return np.abs(z) > threshold

outliers_z = detect_outliers_zscore(X_all[:, 0])
print(f"Z-score 检测异常: {outliers_z.sum()} 个")

# === 机器学习方法 ===
# Isolation Forest
iso_forest = IsolationForest(contamination=0.05, random_state=42)
outliers_iso = iso_forest.fit_predict(X_all) == -1
print(f"Isolation Forest 异常: {outliers_iso.sum()} 个")

# Local Outlier Factor
lof = LocalOutlierFactor(n_neighbors=20, contamination=0.05)
outliers_lof = lof.fit_predict(X_all) == -1
print(f"LOF 异常: {outliers_lof.sum()} 个")

# === 处理异常值 ===
# 1. 删除
X_clean = X_all[~outliers_iso]

# 2. 截断（Winsorize）
from scipy.stats import mstats
X_winsorized = mstats.winsorize(X_all[:, 0], limits=[0.02, 0.02])

# 3. 对数变换（压缩大值）
X_log = np.log1p(np.abs(X_all[:, 0])) * np.sign(X_all[:, 0])

# 4. 用中位数替换
median = np.median(X_all[:, 0])
X_replaced = X_all[:, 0].copy()
X_replaced[detect_outliers_zscore(X_replaced)] = median
```

| 方法 | 假设 | 多维 | 速度 | 适用 |
|------|------|------|------|------|
| IQR | 单变量正态 | 否 | 快 | 单特征 |
| Z-score | 正态分布 | 否 | 快 | 单特征 |
| Isolation Forest | 无 | 是 | 快 | 通用 |
| LOF | 局部密度 | 是 | 慢 | 局部异常 |
| DBSCAN | 密度聚类 | 是 | 中 | 聚类+异常 |

### 8. 完整 Pipeline 示例

```python
import numpy as np
import pandas as pd
from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import StandardScaler, OneHotEncoder, PowerTransformer
from sklearn.impute import SimpleImputer
from sklearn.feature_selection import SelectFromModel
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import cross_val_score

rng = np.random.default_rng(42)

# 构造混合类型数据
n = 500
df = pd.DataFrame({
    'age': rng.normal(40, 12, n),
    'income': rng.exponential(50000, n),
    'score': rng.uniform(0, 100, n),
    'education': rng.choice(['High School', 'Bachelor', 'Master', 'PhD'], n),
    'city': rng.choice(['Beijing', 'Shanghai', 'Guangzhou', 'Shenzhen'], n),
})
df.loc[rng.choice(n, 30), 'age'] = np.nan
df.loc[rng.choice(n, 20), 'income'] = np.nan

y = (df['income'] > df['income'].median()).astype(int)

# 定义数值列和类别列
numeric_features = ['age', 'income', 'score']
categorical_features = ['education', 'city']

# 数值处理 Pipeline
numeric_pipeline = Pipeline([
    ('imputer', SimpleImputer(strategy='median')),
    ('scaler', StandardScaler()),
])

# 类别处理 Pipeline
categorical_pipeline = Pipeline([
    ('imputer', SimpleImputer(strategy='most_frequent')),
    ('encoder', OneHotEncoder(handle_unknown='ignore', sparse_output=False)),
])

# 组合 ColumnTransformer
preprocessor = ColumnTransformer(
    transformers=[
        ('num', numeric_pipeline, numeric_features),
        ('cat', categorical_pipeline, categorical_features),
    ],
    remainder='drop'  # 丢弃未指定的列
)

# 完整 Pipeline
full_pipeline = Pipeline([
    ('preprocessor', preprocessor),
    ('classifier', RandomForestClassifier(n_estimators=100, random_state=42))
])

# 交叉验证
scores = cross_val_score(full_pipeline, df, y, cv=5, scoring='accuracy')
print(f"交叉验证准确率: {scores.mean():.4f} ± {scores.std():.4f}")

# 训练最终模型
full_pipeline.fit(df, y)

# 预测新数据
new_data = pd.DataFrame({
    'age': [35, np.nan],
    'income': [60000, 45000],
    'score': [75, 90],
    'education': ['Master', 'Bachelor'],
    'city': ['Beijing', 'Shanghai'],
})
predictions = full_pipeline.predict(new_data)
print(f"预测: {predictions}")
```

## 面试题

### 1. 特征工程为什么比模型选择更重要？

特征工程决定了模型性能的上限，模型只是在逼近这个上限。原因：（1）模型只能在给定的特征空间中学习，特征中没有的信息无论用什么模型都学不到；（2）好的特征能让简单模型（如线性回归）达到复杂模型的效果，降低过拟合风险；（3）Kaggle 竞赛经验表明，排名靠前的队伍通常特征工程相似，差距在特征构造的细节上。当然，两者都很重要，但优先级应该是：数据质量 > 特征工程 > 模型选择 > 超参调优。

### 2. StandardScaler 和 MinMaxScaler 怎么选？

StandardScaler 将数据变换为均值0、标准差1，假设数据近似正态分布，适合 SVM、逻辑回归、神经网络等基于距离或梯度的模型。MinMaxScaler 将数据缩放到 [0,1]，适合需要输入在固定范围的模型（如神经网络激活函数的敏感区间）。两者对离群值都敏感——一个极端值就能严重拉偏缩放参数。有离群值时用 RobustScaler（基于中位数和 IQR）。实践中，如果没有特殊要求，默认用 StandardScaler。

### 3. 独热编码的高基数问题怎么解决？

当类别变量有很多唯一值（如城市名1000个）时，独热编码会产生1000列稀疏矩阵，导致维度灾难。解决方案：（1）**频率编码**——用类别出现频率替代，1维搞定，适合树模型；（2）**目标编码**——用类别对应的目标变量均值替代，需K-Fold防止泄露；（3）**二值编码**——类别→整数→二进制→拆列，维度从 k 降到 log₂k；（4）**嵌入编码**——用神经网络学习类别的低维表示，适合深度学习；（5）**合并小类别**——频率低于阈值的归为"Other"类。

### 4. 什么是数据泄露？如何在特征工程中避免？

数据泄露是指训练时无意中使用了测试时不可得的信息，导致交叉验证表现虚高、实际部署效果骤降。常见场景：（1）在全部数据上 fit 预处理器（如 StandardScaler），然后才划分训练/测试集——测试集的统计量泄露到训练中；（2）目标编码时在全部数据上计算类别均值；（3）特征选择时使用了全部数据的特征重要性。避免方法：所有 fit 操作只在训练集上进行，transform 应用到测试集；使用 scikit-learn Pipeline 自动管理 fit/transform 边界；目标编码使用 K-Fold 方式计算。

### 5. PCA 的原理是什么？什么情况下不应该用？

PCA 通过正交变换将数据投影到方差最大的方向（主成分），实现降维和去噪。步骤：数据中心化 → 计算协方差矩阵 → 特征分解 → 按特征值排序选前 k 个。不应该用的情况：（1）特征有强非线性关系——PCA 是线性降维，会丢失非线性结构；（2）特征含义重要——PCA 后的特征失去原始含义，无法解释；（3）分类任务中类间方差小于类内方差——PCA 最大化总方差，不区分类别，可能降掉判别性方向（应考虑 LDA）；（4）稀疏数据——PCA 需要中心化，稀疏矩阵变稠密，内存爆炸（用 TruncatedSVD 替代）。

### 6. 过拟合和欠拟合在特征工程中的表现是什么？

过拟合特征工程的表现：训练集准确率很高但验证集差；特征数远超样本数；目标编码没有平滑或交叉验证；多项式特征阶数过高；特征中包含了目标变量的直接变换。欠拟合特征工程的表现：训练集和验证集都差；特征信息不足（如只有原始时间戳没有拆分）；类别变量未编码直接输入；数值尺度差异大未标准化；重要交互特征未构造。解决过拟合：减少特征、增加正则化、使用交叉验证做特征选择。解决欠拟合：增加特征、构造交互项、使用非线性变换。

### 7. 互信息和相关系数有什么区别？

Pearson 相关系数只衡量线性关系，取值 [-1, 1]；互信息衡量任意统计依赖关系（包括非线性），取值 [0, +∞)。例如 y = x²，x 从 -1 到 1 均匀分布时，Pearson r ≈ 0（无线性相关），但互信息 > 0（有确定性关系）。特征选择时：如果怀疑线性关系为主用 F 检验/相关系数；如果可能有非线性关系用互信息；不确定时两者都算，取交集或并集。互信息计算需要分箱或密度估计，小样本时不如相关系数稳定。

### 8. 如何评估特征工程的效果？

三种方式：（1）**单特征评估**——计算特征与目标的互信息/相关系数、特征在树模型中的重要性、单特征模型的性能；（2）**消融实验**——逐步添加/移除特征，观察验证集性能变化，最可靠但最耗时；（3）**Pipeline 交叉验证**——用完整 Pipeline（含预处理+特征选择+模型）做交叉验证，比较不同特征工程方案的分数。注意：不能只看训练集效果，必须看验证集；特征选择本身也要纳入交叉验证循环，否则会过拟合特征选择过程。
