# 📊 项目状态总览

读取 `~/.claude/memory/` 下所有 `项目进度/project_*.md` 文件，输出简洁的状态面板。

## 输出格式

```
📊 项目状态总览

🔴 [项目名] — [一句话状态] — [待办]
🟡 [项目名] — [一句话状态] — [缺什么]
🟢 [项目名] — [一句话状态] — 已完成

---
总计：[N] 个项目 | [M] 个进行中 | [K] 个已完成
```

状态颜色规则：
- 🔴 阻塞/有紧急待办
- 🟡 进行中
- 🟢 已完成/稳定
