# 🧠 会话启动 — 读取记忆

每次新会话开始时，执行以下流程：

## 1. 读取全局记忆

读取 `~/.claude/memory/MEMORY.md` 索引，然后读取以下核心文件：

- `user_profile.md` — 用户是谁、项目仓库、基础设施
- `习惯分析/habit_communication.md` — 沟通风格
- `习惯分析/habit_work_pattern.md` — 工作模式
- `习惯分析/habit_decision_pattern.md` — 决策偏好
- `自我反思/reflection_latest.md` — 上次会话的反思
- `自我反思/lesson_learned.md` — 经验教训

根据当前项目目录，读取相关的 `项目进度/project_*.md`。

## 2. 分析用户当前状态

根据记忆内容，判断：
- 用户当前在哪个项目？
- 上次做到哪了？（项目进度）
- 上次被纠正了什么？（反思）
- 有什么待办未完成？

## 3. 输出会话准备摘要

用简洁格式输出：

```
👤 用户：[身份] | 当前项目：[项目名]
📊 上次进度：[一句话]
⚡ 待办/阻塞：[如有]
🪞 上次反思：[一句话]
🎯 本次建议：[基于状态的建议]
```

## 4. 调整协作策略

根据习惯分析自动调整：
- 沟通风格 → 匹配用户的简洁/直接偏好
- 工作模式 → 如果用户集中推进，一口气完成，不碎片化
- 决策偏好 → 方案推荐偏向实用/稳定
