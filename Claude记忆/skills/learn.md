# 📝 快速学习 — 记录发现

用于即时记录一个新发现。格式：

`/learn [内容]`

例如：
- `/learn 用户不喜欢用 CSS-in-JS`
- `/learn FastAPI 的依赖注入比中间件更适合做权限校验`
- `/learn 用户做知识库时喜欢先写索引再填内容`

## 执行流程

1. 解析用户输入的发现内容
2. 判断属于哪个类别：
   - 用户偏好/习惯 → 更新 `习惯分析/habit_*.md` 或 `用户偏好/feedback_*.md`
   - 踩坑/技术经验 → 追加到 `自我反思/lesson_learned.md`
   - 项目信息 → 更新对应的 `项目进度/project_*.md`
   - 新项目 → 创建新的 `项目进度/project_*.md`
3. 写入记忆文件
4. 运行同步

## 输出

```
✅ 已记录到 [文件名]
📝 [内容摘要]
```
